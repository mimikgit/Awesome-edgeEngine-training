üìö Training Overview

Welcome to an immersive 3-day hands-on training course with mimik technology. Day 1 focuses on understanding the principles of Hybrid Edge Cloud (HEC) and setting up the development environments and tools. Day 2 delves into applying developer tools and analyzing edgeEngine functionality. Day 3 culminates in evaluating Hybrid Edge Cloud solutions and applying your understanding by establishing discovery, connection, and communication between edgeEngine-enabled devices. This comprehensive exploration of HEC, edgeEngine, and essential tools paves the way for effective communication with edgeEngine-enabled devices.

üìã Agenda

Day 1: Introduction and Setup (2 hours)

    By the end of Day 1, you will be able to:

  • (15 mins) Understand the concepts of Hybrid Edge Cloud & edgeEngine.
  • (15 mins) Create a mimik account in the mimik developer console.
  • (15 mins) Create an edge project in the mimik developer console.
  • (15 mins) Download and set up third-party development tools.
  • (20 mins) Download and run edgeEngine
  • (20 mins) Set up the mimik development environment & edge-cli tool.
  • (20 mins) Complete the Day 1 review and knowledge check.

Day 2: Developer Tools & hands-on practice (2 hours)

  • (20 mins) Assess the development setup and ensure edgeEngine is up and running properly.
  • (20 mins) Run mDebug on their machines.
  • (40 min) Complete guided exercises for hands-on practice with the mDebug developer tool.
  • (40 min)Complete a Day 2 review and knowledge check

Day 3: Hands-on Activity: mimik JukeBox and API Calls (2 hours)

  • (30 mins) Work with mimik JukeBox.
  • (30 mins) Access JukeBox through API calls.
  • (40 mins) Complete a final project that integrates the skills and knowledge acquired during the training.
  • (20 mins) Participate in a wrap-up session to reflect on the learning experience.
